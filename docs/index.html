Daily tech & gadget videos · @thegadgetcity_01

This is the home page for a personal content scheduling tool that posts daily tech and gadget videos to @thegadgetcity_01 on TikTok and @the_gadget_city_ on Instagram.

The tool is open source on GitHub. It generates one short video per day using OpenAI for the script, image, and voiceover, composes the video with ffmpeg, and uploads it to the operator's drafts inbox via TikTok's official Content Posting API.
