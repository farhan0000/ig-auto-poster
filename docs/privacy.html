Privacy Policy

Last updated: April 2026

The Gadget City ("the project", "we", "us") is a personal, non-commercial content scheduling tool. This privacy policy explains what data the project handles.

1. Data we collect

None from third parties. The project is for personal single-account use. We do not collect, store, or process personal data from any other users.

2. Data the project handles

  • OAuth access tokens for the operator's own TikTok account, stored as encrypted GitHub Actions secrets.
  • The operator's own OpenAI API key, stored the same way.
  • Generated video content (text, images, voiceover, mp4 files), stored in the project's GitHub repository.

None of this data is shared with any third party other than the services strictly required to perform its function (OpenAI for content generation, TikTok for posting).

3. Cookies, analytics, advertising

This website does not use cookies, analytics, or advertising of any kind.

4. Third-party services

  • OpenAI (text, image, text-to-speech generation) — privacy policy
  • TikTok Developer Platform (content posting API) — privacy policy
  • GitHub (hosting, scheduled actions) — privacy statement

5. Contact

Questions or concerns: open an issue at the GitHub repository.
